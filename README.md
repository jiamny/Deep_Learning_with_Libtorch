# C++ Implementation of PyTorch Tutorial projects

Collected deep learning projects with LibTorch C++

## Dependency
- cmake 3.14+
- PyTorch 1.9.1
- python3.6+
- opencv 4.3.0
- matplotlib-cpp
- ubuntu 18.04
